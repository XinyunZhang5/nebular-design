Nebular Design — 项目框架

Requirements
用户能上传建筑图片，系统将其转化为 3D 数据，匹配 LEGO 积木样式，生成组装步骤，并提供社交功能让用户交流互动。

User Stories

作为用户，我想上传建筑图片，得到对应的 LEGO 积木清单和组装步骤
作为用户，我想看到每块积木对应 LEGO 官方商店的真实款式
作为用户，我想在 Chat Room 里找到有共同爱好的人
作为用户，我想加好友，和特定用户保持长期联系
作为用户，我想看到我历史上传过的作品记录


Acceptance Criteria

上传图片后 X 秒内返回积木清单
积木样式必须来自 LEGO 官方已有款式
组装步骤以分步图文形式展示
Chat Room 支持实时消息
好友系统支持发送/接受/拒绝请求
用户需注册登录才能使用核心功能

Architecture

    ↓
Next.js 前端 (Vercel)
    ↓ API 请求

    ├── 图片处理服务（2D→3D）
    ├── AI 积木匹配服务

API Design
图片处理
POST   /api/images/upload        上传图片
GET    /api/images/{id}/status   查询转换进度
GET    /api/images/{id}/result   获取积木清单+步骤
用户系统
POST   /api/auth/register        注册
POST   /api/auth/login           登录
GET    /api/users/{id}           获取用户信息
好友系统
POST   /api/friends/request      发送好友请求
POST   /api/friends/accept       接受请求
GET    /api/friends/list         好友列表
聊天
GET    /api/chatroom/messages    获取历史消息
WS     /ws/chatroom              实时聊天 WebSocket
WS     /ws/dm/{friend_id}        私信 WebSocket

ERD
User
├── id
├── username
├── email
├── avatar_url
└── created_at

Friendship
├── id
├── requester_id → User
├── receiver_id  → User
└── status (pending/accepted)

Project（作品）
├── id
├── user_id      → User
├── image_url
├── result_json  （积木清单+步骤）
└── created_at

Message（Chat Room）
├── id
├── sender_id    → User
├── content
├── type (public/dm)
├── receiver_id  → User (私信时用)
└── created_at

Schema
sqlCREATE TABLE users (
  id UUID PRIMARY KEY,
  username VARCHAR(50) UNIQUE NOT NULL,
  email VARCHAR(255) UNIQUE NOT NULL,
  avatar_url TEXT,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE friendships (
  id UUID PRIMARY KEY,
  requester_id UUID REFERENCES users(id),
  receiver_id UUID REFERENCES users(id),
  status VARCHAR(20) DEFAULT 'pending'
);

CREATE TABLE projects (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  image_url TEXT NOT NULL,
  result_json JSONB,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE messages (
  id UUID PRIMARY KEY,
  sender_id UUID REFERENCES users(id),
  receiver_id UUID REFERENCES users(id),
  content TEXT NOT NULL,
  type VARCHAR(10) DEFAULT 'public',
  created_at TIMESTAMP DEFAULT NOW()
);
